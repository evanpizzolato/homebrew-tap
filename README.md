# homebrew-tap

Homebrew casks for [Evan Pizzolato](https://github.com/evanpizzolato)'s apps.

## Usage

```bash
brew install --cask evanpizzolato/tap/ghostwriter
```

## Casks

| Cask | Description |
|------|-------------|
| [`ghostwriter`](Casks/ghostwriter.rb) | [Ghostwriter](https://evanpizzolato.github.io/ghostwriter/) — privacy-focused notes that stay hidden from screenshots and screen shares |
